# 🎮 技能拾取MOBA — 忍者密卷之战

> 一款移动端MOBA游戏，融合《火影忍者》主题 + 《王者荣耀》推塔对抗 + 《PUBG》拾取/掉落机制。
> 英雄不自带技能，所有技能通过打野拾取获得，死亡时掉落。

---

## ⚡ 技术概览

| 维度 | 说明 |
|------|------|
| **引擎** 🛠️ | Godot 4.7.1（GDScript），纯 2D 渲染管线，Y-sort 实现 2.5D 俯视视角 |
| **平台** 📱 | Android（APK 导出），含 PC 键盘调试模式 |
| **架构** 🏗️ | 场景树驱动 + 信号总线，CanvasLayer 独立 UI 层（`layer=10`），逻辑与表现分离 |
| **游戏循环** 🔄 | AI 状态机（打野→寻敌→战斗→拆塔）+ 物理碰撞帧同步 |
| **输入系统** 🕹️ | 自定义虚拟摇杆（`_input` 全局事件） + GUI 按钮（`gui_input` 信号），双端兼容 |
| **渲染** 🎨 | `Sprite2D` 贴图 + `_draw()` 降级兜底，chroma-key 去背，y轴 sin 浮动动画 |
| **碰撞** 💥 | `Area2D` + `PhysicsRayQueryParameters2D` 射线检测，直接位移击退 |

### 🏆 核心架构亮点

- **🧩 纯 GDScript 无插件依赖**，零 C++ 模块，场景树即架构图
- **🧠 AI 行为树**：三态轮询状态机（FARMING → SEEKING → FIGHTING），每帧 `_process` 驱动，无协程调度
- **⚙️ 技能系统**：`SkillManager` 统一调度 + 独立场景脚本（`Fireball.gd` / `RasenganArea.gd` / `ShadowClone.gd`），热插拔设计
- **🔗 Unity-Style 分组**：Godot 组系统（`human_player` / `monsters` / `bases` / `camps`）替代硬编码引用，支持运行时动态注册
- **📲 移动端适配**：`CanvasLayer` 独立 UI + `call_deferred` 延迟初始化 + 视口等比缩放，适配 360px ~ 1920px 屏幕

## 📋 游戏设计总纲

### 💡 一句话核心
> 🥷 忍者成长，打败对应的怪，学习新的密卷。开局空装 → 打野拾取技能 → 遭遇战 → 击杀/死亡掉落技能 → 循环。

### 🌍 世界观
| 项目 | 内容 |
|------|------|
| **风格** 🎯 | 火影忍者 |
| **地图** 🏞️ | 木叶村周边森林 |
| **阵营** ⚔️ | 宇智波一族 vs 千手一族 |
| **美术** 🎨 | 火影动漫风（AI生成素材） |
| **目标用户** 🎮 | 硬核PVP玩家，钟爱博弈 |

### ⚔️ 核心规则
- **👥 人数**: 1v1（先做AI对手，后续联机）
- **🗺️ 地图**: 一条横向直线，左右两端各一个基地
- **👹 野怪**: 3个营地（左怪、中Boss、右怪），击败随机掉落技能
- **📦 技能**: 3个技能槽，从野怪拾取，死亡掉落随机1个
- **🏆 胜利条件**: 摧毁对方基地（基地HP 20格，每60秒自动+5最大生命值）
- **🚫 无**金币、无等级、无蓝条、无草丛、无兵线、无防御塔

## 🧍 英雄设计

> 🎲 **角色随机分配**：开局50%概率 玩家=宇智波佐助/敌方=千手鸣人，50%概率 玩家=千手鸣人/敌方=宇智波佐助。
> 贴图、连击风格、专属技能随分配自动切换。

### 🔥 宇智波一族（参考佐助 🗡️）
| 属性 | 值 |
|------|:---:|
| **❤️ 生命值** | 10格 |
| **⚔️ 普攻** | 近战，攻速**快**，每次 0.5 格伤害 |
| **💨 第4次普攻** | **位移突进**（前冲150px），带无敌帧+穿墙效果 |
| **❓ 被动** | 暂无 |
| **🔥 专属技能** | 火球术**发射4个**（普通3个），居中排列 |

### 💨 千手一族（参考鸣人 🦊）
| 属性 | 值 |
|------|:---:|
| **❤️ 生命值** | 10格 |
| **⚔️ 普攻** | 近战，攻速**中**（0.75s间隔），每次 **0.5** 格伤害 |
| **💥 第3次普攻** | **击退**，附带前冲50px，期间目标不可攻击、不可使用密卷 |
| **❓ 被动** | 暂无 |
| **🌀 专属技能** | 螺旋丸**范围翻倍**（scale=2） |

## 🔥 技能系统

### 🎯 三大初始技能

| 技能 | 效果 | 冷却 ⏱️ | 类型 |
|:---:|------|:---:|:----:|
| **🔥 火球术** | 向前方发射火球（千手3个/宇智波4个），中距离；距离越近击退越远，伤害 **3~4格**（单个1格） | 5s | 伤害+击退 |
| **🌀 螺旋丸** | 短位移突进 + 小范围持续伤害 3s，每秒 **1格** 伤害 | 4s | 位移+范围伤害 |
| **👥 影分身** | 制造一个分身跟随在身后；**未受伤** → 下次普攻或密卷**效果翻倍**后消失（如火球变6个）；**受伤** → 分身吸收**最多2点**伤害后消失。分身**不因施法消失**，仅在被击或吸收满伤害时消失 | 10s | 防御+强化 |

### 📜 技能规则
- 🎲 技能**不分稀有度**，所有野怪随机掉落三种技能之一
- 🔄 可捡到**重复技能**（两个火球术可同时存在）
- ⬆️ **不可升级**，不可合成
- ❌ **无技能组合效果**
- ⏳ 仅消耗**冷却时间**，无MP/能量消耗

### 💎 拾取规则
| 来源 | 说明 |
|------|------|
| 💀 击杀野怪 | 随机掉落一个技能 |
| 👑 击杀Boss | 掉落**两个技能** + **❤️ 红色生命球**（拾取后永久+5最大HP） |
| ⚔️ 击杀英雄 | 敌方死亡后**随机掉落一个**技能到地面 |
| ⏰ 刷新时间 | 普通野怪 **40s**，Boss **80s** |
| ⌛ 生命球消失 | 30秒无人拾取自动消失 |

## 🗺️ 地图布局

### 📍 三营地直线分布

```
  [宇智波基地] ———— 左营地(100,600) ———— 中Boss(450,300) ———— 右营地(1000,600) ———— [千手基地]
  (80,750)            (普通野怪)           (Boss)              (普通野怪)              (1420,750)
```

### 📐 地图参数
| 参数 | 数值 |
|------|:----:|
| 📏 地图尺寸 | 1500 × 1500 像素 |
| 🛤️ 主路径 | 横向直线，y=680~760 |
| 🏃 角色速度 | 100px/s（1单位/秒），AI为玩家0.9倍（90px/s） |
| 📍 相邻营地间距 | 左→中约350px，中→右约550px |
| 🧱 边界 | 有边界墙，不可越界 |
| 障碍物 | 无石墙掩体 |

### 🏰 基地
| 项目 | 内容 |
|------|------|
| 🏯 外观 | 火影村庄建筑 |
| ❤️ 基础血量 | **20格** |
| 📈 自动成长 | **每60秒**增加5点最大生命值（随时间推进越来越难拆） |
| 🩹 治疗区域 | 靠近己方基地（HealArea半径200px），每秒回复 **2格** HP |
| 🚨 反击机制 | 基地5s内累计受伤≥5点，AI立即放弃一切行为追击攻击者（最高优先级） |
| 🏆 胜利条件 | 摧毁对方基地 |

### 💀 死亡与复活
| 项目 | 内容 |
|------|------|
| ⏳ 复活时间 | **10秒** |
| 📍 复活位置 | 己方基地 |
| 💧 死亡掉落 | **随机1个技能**掉落地面（其余保留） |
| 🛡️ 无敌时间 | 复活后立即恢复行动，无额外无敌时间 |
