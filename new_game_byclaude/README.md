# 🎮 技能拾取MOBA — 忍者密卷之战

> 一款移动端MOBA游戏，融合《火影忍者》主题 + 《王者荣耀》推塔对抗 + 《PUBG》拾取/掉落机制。
> 英雄不自带技能，所有技能通过打野拾取获得，死亡时掉落。

---

## 📋 游戏设计总纲

### 一句话核心
> 忍者成长，打败对应的怪，学习新的密卷。开局空装 → 打野拾取技能 → 遭遇战 → 击杀/死亡掉落技能 → 循环。

### 世界观
| 项目 | 内容 |
|------|------|
| **风格** | 火影忍者 |
| **地图** | 木叶村周边森林 |
| **阵营** | 宇智波一族 vs 千手一族 |
| **美术** | 火影动漫风（AI生成素材） |
| **目标用户** | 硬核PVP玩家，钟爱博弈 |

### 核心规则
- **人数**: 1v1（先做AI对手，后续联机）
- **地图**: 一条横向直线，左右两端各一个基地
- **野怪**: M形分布的5个营地，击败随机掉落技能
- **技能**: 3个技能槽，从野怪拾取，死亡掉落随机1个
- **胜利条件**: 摧毁对方基地（基地HP 20格）
- **无**金币、无等级、无蓝条、无草丛、无兵线、无防御塔

| **被动** | 暂无 |

| 属性 | 值 |

| **被动** | 暂无 |
| **专属技能** | 暂无 |

## 🔥 技能系统

### 三大初始技能

| 技能 | 效果 | 冷却 | 类型 |
|:---:|------|:---:|:----:|
| **火球术** | 向前方发射三个火球，中距离；距离越近击退越远，伤害 **3格**（3×1） | 5s | 伤害+击退 |
| **螺旋丸** | 短位移突进 + 小范围持续伤害 2s，每秒 **1格** 伤害 | 4s | 位移+范围伤害 |
| **影分身** | 制造一个分身跟随在身后；若未受伤，下一次普攻或密卷**效果翻倍**后消失（如火球变6个）；若受伤，分身吸收**最多2点**伤害后消失 | 10s | 防御+强化 |

### 技能规则
- 技能**不分稀有度**，所有野怪随机掉落三种技能之一
- 可捡到**重复技能**（两个火球术可同时存在）
- **不可升级**，不可合成
- **无技能组合效果**
- 仅消耗**冷却时间**，无MP/能量消耗

### 拾取规则
| 来源 | 说明 |
|------|------|
| 击杀野怪 | 随机掉落一个技能（Boss掉落**两个**） |
| 击杀英雄 | 敌方死亡后**随机掉落一个**技能到地面 |
| 刷新时间 | 野怪死亡后 **40s** 刷新 |

## 🗺️ 地图布局

### M形野怪分布

```
                   营地2(400,400)         营地4(1100,400)
                      \                   /
                       \                 /
  [宇智波基地] ———— 营地1(200,580) ———— 营地3(750,300) ———— 营地5(1300,580) ———— [千手基地]
  (80,750)             /   (Boss)    \             (1420,750)
                      /                 \
                 营地1               营地5
```

### 地图参数
| 参数 | 数值 |
|------|:----:|
| 地图尺寸 | 1500 × 1500 像素 |
| 主路径 | 横向直线，y=680~760 |
| 角色速度 | 100px/s（1单位/秒） |
| 相邻营地间距 | 约 3 单位 |
| 边界 | 有边界墙，不可越界 |
| 障碍物 | 野怪区域有石墙掩体 |

### 基地
| 项目 | 内容 |
|------|------|
| 外观 | 火影村庄建筑 |
| 血量 | **20格** |
| 保护机制 | 无 |
| 胜利条件 | 摧毁对方基地 |

### 死亡与复活
| 项目 | 内容 |
|------|------|
| 复活时间 | **10秒** |
| 复活位置 | 己方基地 |
| 死亡掉落 | **随机1个技能**掉落地面（其余保留） |
| 无敌时间 | 无 |

## 🚀 快速开始

1. 打开 **Godot 4.7.1**（位于 `D:\gotot\Godot_v4.7.1-stable_win64.exe`）
2. 点击 **Import** → 选择本项目文件夹
