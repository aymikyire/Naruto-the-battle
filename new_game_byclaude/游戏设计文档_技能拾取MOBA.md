## 一、核心概念

### 一句话概括
> 一款**移动端MOBA游戏**，但英雄不自带技能，所有技能通过地图中**拾取获得**，死亡时掉落，形成"搜集—战斗—风险"的循环。融合了《王者荣耀》的推塔对抗 + 《PUBG》的拾取/掉落机制。

### 核心循环
```
开局空装 → 打野/清兵 → 拾取技能 → 遭遇战 → 击杀/死亡
                                  ↑              ↓
                              掉落/更换 ←── 死亡掉落技能
```

### 核心乐趣来源
| 乐趣点       | 说明                        |
| --------- | ------------------------- |
| 随机性与策略的平衡 | 每局捡到的技能不同，需要根据"捡到什么"来调整打法 |
| 高风险高回报    | 身上技能越好越强，但死了掉得也越痛         |
| 刷野路线规划    | 类似吃鸡跳伞后的搜集路线，需要决策         |
| 技能组合的化学反应 | 不同技能搭配能产生意想不到的combo       |

> [?] 你觉得哪些类型是必须有的？哪些可以砍掉？
你的技能太丰富了，现在我自己设计的技能只有三个，其他详细技能游戏成功跑通之后再添加。
1.火球术:向前方发射三个火球，中距离，若释放火球时对手距离自己距离越近则击退越远，反之。（角色生命预设为5格）该技能伤害为2格，冷却时间5s
2.螺旋丸：角色短暂向前位移一段距离，释放螺旋丸，造成小范围内的持续伤害，持续时间2s,每秒1格伤害，冷却时间4s
3.影分身之术:使用技能后进行分身出两个角色，若此时未受到伤害，则接下来的普攻或者技能则会造成双倍伤害，双倍特效效果。若受到伤害>2，则影分身消失，换言之，可以为角色抵挡2格伤害，并消失。冷却时间10s
其余技能暂时不设置
