## 第一阶段：核心战斗原型

> 目标：能在一条路上走、打野、捡技能、放技能、打死对方

| 模块 | 具体内容 | 预估难度 |
|:----|---------|:-------:|
| **基础场景** | 一条直线的地图 + 两个端点 + 地面贴图 | ⭐ |
| **角色控制器** | 移动（摇杆/键盘）、2.5D碰撞 | ⭐⭐ |
| **普攻系统** | 攻击判定、伤害计算（每格血） | ⭐⭐ |
| **技能系统** | 3个技能：火球术、螺旋丸、影分身 | ⭐⭐⭐ |
| **野怪系统** | 5个野怪营地、击败掉落技能 | ⭐⭐ |
| **技能拾取** | 走到掉落物上拾取、替换已有技能 | ⭐⭐ |
| **技能栏UI** | 显示3个技能槽、CD倒计时 | ⭐⭐ |
| **血条UI** | 5格血显示 | ⭐ |
| **死亡与复活** | 死亡掉落所有技能、基地复活 | ⭐⭐ |
| **基地摧毁** | 胜利条件判定 | ⭐ |
